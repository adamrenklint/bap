## API

### bap

- ```bap.kit()``` returns a new [kit](#kit)
- ```bap.slot()```
- ```bap.layer()
- ```bap.pattern()```
- ```bap.sequence()```
- ```bap.channel()```
- ```bap.note()```

### kit

- ```kit.slot()``` returns blank [slot](#slot) assigned to next index
- ```kit.slot(index)``` returns existing or blank slot with *index*
- ```kit.slot(index, slot)``` set existing *slot* at *index*
